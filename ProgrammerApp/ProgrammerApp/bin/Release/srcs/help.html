Uploading

  • Uploading is performed completely by simply pressing upload. For successful upload, each programmer needs to be connected to a board via the ribbon cable.
    1. Plug in one circuit card to each programmer.
    2. Plug in each programmer, refresh if needed.
    3. A status box will appear at the bottom for each progammer connected displaying "Board Connected".
    If not, refer below.
    4. Click "Program" and wait for completion.
    If not successful, refer below.
    5. Otherwise, programming is complete.

Uploading Error

  • The uploading process performs up to 5 attempts if an error occurs. The programmer status will display red with with the first failure and change if successful. If all 5 attempts fail, check connections. If the failure continues, the board is most likely faulty.

Connecting Programmers

  • USB Programmers should connect automatically and list at the bottom of the application. If programmers are connected and do not appear, click "Refresh". If the problem persists, attempt to re-locate the programmer driver

Selecting Hex

  • The programming process uploads the source code required for each board as a ".hex" file. In order to locate the correct ".hex" file, click "Locate Hex", navigate to the desired ".hex" and click "Open".

  • Back to Top

Locating Programmer Driver

  • If the application is not recognizing connected programmers, click "Locate Driver" and navigate to a valid "avrdude.exe" type driver file.
